BGlib.gmode.translators.df_utils.virtual_data.VirtualDataset¶

class BGlib.gmode.translators.df_utils.virtual_data.VirtualDataset(name, data, dtype=None, compression=None, chunking=None, parent=None, resizable=False, maxshape=None, attrs=None)[source]¶

Bases: BGlib.gmode.translators.df_utils.virtual_data.VirtualData

Holds data (i.e. numpy.ndarray) as well as instructions on writing, attributes, etc… This gets converted to a h5py.Dataset by io.HDFwriter

Region references need to be specified using the ‘labels’ attribute. See example below

Parameters

  • name (String) – Name of the dataset

  • data (Object) – See supported objects in h5py

  • dtype (datatype,) – typically a datatype of a numpy array =None

  • compression ((Optional) String) – See h5py compression. Leave as ‘gzip’ as a default mode of compression

  • chunking ((Optional) tuple of ints) – Chunking in each dimension of the dataset. If not provided, default chunking is used by h5py when writing this dataset

  • parent ((Optional) String) – HDF5 path to the parent of this object. This value is overwritten when this dataset is made the child of a datagroup. Default = under root

  • resizable (Boolean (Optional. default = False)) – Whether or not this dataset is expected to be resizeable. Note - if the dataset is resizable the specified maxsize will be ignored.

  • maxshape ((Optional) tuple of ints) – Maximum size in each axis this dataset is expected to be if this parameter is provided, io will ONLY allocate space. Make sure to specify the dtype appropriately. The provided data will be ignored

  • attrs (dict (Optional) Default = None) – Attributes to be attached to the h5py.Dataset object

Examples

  1. Small auxiliary datasets :

    Make sure to specify the name and data. All other parameters are optional

>>> ds_ex_efm = VirtualDataset('Excitation_Waveform', np.arange(10))

  1. Initializing large primary datasets of known sizes : See EmptyVirtualDataset

  2. Initializing large datasets whose size is unknown in one or more dimensions: See ExpandableVirtualDataset

  3. Datasets with region references :

>>> ds_spec_inds = VirtualDataset('Spectroscopic_Indices', np.random.random(1, 10))
    ds_spec_inds.attrs['labels'] = {'Time Index':(slice(0,1), slice(None))}
